# Smart AI Cover Letter Generator

A modern, fast, and responsive web application built with **Next.js** and powered by **Google Gemini AI**. This tool helps developers and professionals instantly generate highly tailored, professional cover letters by analyzing their resume and the target job description.

## 🚀 Live Demo
**[Click here to view the live project](https://ai-cover-letter-nu-nine.vercel.app/)**

## 🛠️ Tech Stack
- **Frontend Framework:** Next.js, React
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **AI Integration:** Google Gemini AI API

## 💡 Features
- Instant cover letter generation tailored to specific job requirements.
- Clean, minimalist, and responsive user interface.
- Easy copy-to-clipboard functionality.
